# v1.2.7 — Business Validation Layer

This release adds a deterministic business-readiness layer for batch output.

It separates artifact creation from downstream coding readiness:

```text
technical_success  -> code completed and artifact row exists
scrape_success     -> Crawl4AI captured real product-page evidence
evidence_success   -> product evidence is sufficiently grounded
visual_success     -> clean product image evidence is present
```

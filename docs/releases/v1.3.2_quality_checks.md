# v1.3.2 — Local Quality Checks

This release adds local quality-check commands to make validation faster before running or merging changes.

## New commands

Compile all Python files:

```bash
pdm run check-compile
```

Run tests:

```bash
pdm run check-tests
```

Run both:

```bash
pdm run quality-check
```

## What this catches

```text
syntax errors
broken imports in package modules
broken scripts
failing pytest cases
```

This release does not change scraping behavior and does not add search. It only adds local validation commands.
